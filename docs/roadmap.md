# Roadmap

## Phase 0 — Manual prototype

Use ChatGPT to generate weekly Moon & Table briefs before building too much.

Test:

* one theme
* one best window
* one ritual
* one prompt
* one optional add-on
* under 20 minutes by default

Capture feedback:

* meaningful
* too cheesy
* too much
* too generic
* do again
* more plant / candle / kitchen / astrology / numerology
* lower capacity needed

## Phase 1 — Static weekly brief MVP

Build a simple private web app that can show one generated weekly brief.

Features:

* private login
* current week view
* theme
* best window
* recommended ritual
* relationship prompt
* optional add-on
* “why this” explanation
* save/dismiss feedback

## Phase 2 — Curated symbolic library

Create editable library entries for:

* 8 moon phases
* 12 zodiac signs
* planets/luminaries
* major aspects
* retrogrades
* numerology 1–9
* common herbs/spices
* candle colors
* kitchen magic actions
* plant magic actions
* seasonal themes

Each entry should include:

* key
* type
* themes
* good for
* avoid saying
* ritual styles
* source notes
* confidence/mainstreamness
* safety notes where needed

## Phase 3 — Personal profiles

Add private profiles for Tim and Jessica.

Profile fields may include:

* birth date/time/place
* natal placements
* preferred ritual styles
* disliked ritual styles
* capacity defaults
* max ritual duration
* preferred nights
* important personal dates
* schedule constraints

Initial Jessica themes:

* Pisces Sun
* Virgo Moon
* Leo Rising
* Mercury in Aquarius
* Venus in Aries
* Mars in Capricorn
* Jupiter in Aries
* Saturn in Capricorn

## Phase 4 — Schedule-aware planning

Add manually defined schedule rules first.

Rules:

* unavailable nights
* preferred windows
* parenting schedule
* school/work constraints
* max ritual duration
* default low-capacity mode

Later add calendar integration:

* Google Calendar
* possibly Apple Calendar
* detect realistic windows
* avoid busy nights
* move ritual suggestions near symbolic events, not necessarily on the exact date

## Phase 5 — Computed timing engine

Add reliable computed data:

* moon phase
* new/full moons
* solstice/equinox
* lunar cycles from personal dates
* universal numerology days
* personal numerology dates

Later:

* moon sign
* planetary ingresses
* retrogrades
* aspects
* personal transits

## Phase 6 — Ritual notebook

Add a private saved history:

* rituals done
* notes
* what worked
* what felt off
* repeat this
* seasonal favorites
* relationship prompts worth revisiting

## Phase 7 — Curation workbench

Add admin tools for managing source material:

* source inbox
* manual excerpt/note entry
* classify source
* summarize into atomic cards
* approve/edit before library inclusion
* maintain source notes

## Not now

Do not build these early:

* public sharing
* community rituals
* marketplace/content subscriptions
* complex predictive astrology
* full birth chart engine
* daily notification spam
* large-scale web scraping
* AI-generated meanings without curated grounding
