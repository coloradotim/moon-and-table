# Source gate record: Moon Book

## Gate verdict

`approved_for_extraction_with_limits`

*The Moon Book* is a strong Moon & Table source for future extraction, especially for lunar-cycle structure, phase-based ritual mechanics, journaling, spell/ritual architecture, altar practice, moon-water/vessel work, candle/moon timing, and reflective moontime practice.

Approval is limited because this is a modern copyrighted source with highly authored language, spells, prompts, meditations, correspondences, affirmations, and worldview-specific framing. Runtime use is not approved.

Broadly approved for future extraction:

- lunar phase structure;
- new moon / waxing / full / waning / dark moon mechanics;
- lunar journaling and cycle tracking;
- intention, release, blessing, gratitude, protection, reflection;
- moon altar, candle, vessel/water, words, and body-based mechanics.

Limited or excluded:

- exact spells, rituals, prompts, meditations, affirmations, prayers, invocations, and distinctive phrasing;
- medical, therapeutic, trauma-processing, fertility, menstrual, healing, addiction, mental-health, or guaranteed-effect claims;
- deity/spirit/ancestor material without separate approval;
- cultural/living-practice material without provenance review;
- herb/crystal/oil/correspondence material without safety filtering.
