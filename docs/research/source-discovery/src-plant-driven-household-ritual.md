## Scope

This report identifies source candidates for Moon & Table’s plant-driven household ritual gap.

This is source discovery only. It does not run source gate, does not extract ritual candidates, does not create runtime Ritual records, does not update implementation, and does not mark any source approved.

## Plant gap being researched

Moon & Table needs plant-as-primary-carrier ritual sources, especially:

- `plant + opening`
- `plant + voicing`
- `plant + remembering`
- `plant + steadying`
- `plant + marking`
- `plant + blessing`
- `plant + protecting`
- `plant + releasing`

The desired source profile is household-scale, domestic, private, warm, source-backed, and practice-rich. The strongest sources should provide plant ritual mechanics, not only plant correspondences.

## Review depth

Discovery-level public metadata and bibliographic review only. No source copy was reviewed. Publication details, TOCs, page ranges, and exact mechanics must be verified during source gate from lawful source copies.

## Evaluation standard

For Moon & Table, “plant-driven” means the plant is the ritual carrier, witness, offering, living participant, seasonal marker, household threshold, or memory object. A source is weaker if plants appear only as ingredients in oils, incense, teas, sachets, or correspondence lists.

Caution flags:

- medicinal or herbal-health claims;
- ingestion, smoke, essential oil, tincture, or topical recipes;
- toxic plants, pets/children/pregnancy/allergy concerns;
- outdoor gardening that does not translate to household/private ritual;
- plant-spirit, deity, or culturally specific work that requires boundary review;
- coercive protection/banishing or enemy-focused plant magic;
- correspondence-only encyclopedias with little ritual sequence.
