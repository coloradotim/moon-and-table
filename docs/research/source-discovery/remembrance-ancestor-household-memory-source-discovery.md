# Source discovery report: remembrance, ancestors, beloved dead, and household memory

## Scope

Issue: #297

This is source discovery only. It does not run source gate, extract Ritual candidates, create Ritual records, update runtime data, fill a coverage matrix, or create implementation work.

This lane is optional. The goal is not to assume ancestor work belongs in Moon & Table. The goal is to identify whether there are source candidates that can support private, grounded, domestic remembrance without turning the app into grief therapy, ancestor religion, spirit communication, or divination.

## Discovery posture

The four-packet audit left several remembering cells empty, thin, or fragile, especially:

```text
doorway + remembering
plant + remembering
body + remembering
vessel + remembering
```

Moon & Table may need a source-backed lane for:

- household memory ritual;
- remembered homes and seasons;
- beloved-dead remembrance;
- memory objects, vessels, altars, plants, candles, doors, and tables;
- intimate private remembrance that stays warm and ritual-shaped;
- optional ancestor material only if it fits Tim and Jessica and stays source-boundaried.

Good source candidates should support household remembrance rather than broad ancestor religion, grief counseling, mediumship, or divination.
