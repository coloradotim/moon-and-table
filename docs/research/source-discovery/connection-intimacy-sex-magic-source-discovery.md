# Source discovery report: connection, intimacy, and sex-magic lane

## Scope

Issue: #293

This is source discovery only. It does not run source gate, extract Ritual candidates, create Ritual records, update runtime data, fill a coverage matrix, or create implementation work.

The research target is source-backed material for consensual connection, intimacy, erotic energy, desire, devotion, mutual attention, two-person practice, shared candle ritual, shared vessel/cup/bowl/object ritual, touch, witnessing, repair, invitation, closeness, household altar/table/bedroom practices, and private adult partner ritual.

## Discovery posture

The four-packet coverage audit identified connection as a remaining gap, especially:

```text
candlelight + connecting
vessel + connecting
```

The lane should not default to generic love spells. Historically and in modern occult publishing, love magic often includes attraction, compulsion, control, jealousy, fertility, and target-specific desire work. Moon & Table should prefer sources that can support mutual, consent-aware, partner-facing, household-scale practices.
