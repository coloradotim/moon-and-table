# Source discovery report: vessel, cup, bowl, and shared-object ritual

## Scope

Issue: #296

This is source discovery only. It does not run source gate, extract Ritual candidates, create Ritual records, update runtime data, fill a coverage matrix, or create implementation work.

The research target is source-backed material for vessels and shared objects as relational, holding, blessing, memory, witnessing, and household ritual carriers. The point is not more protection jars. Moon & Table already has vessel support for releasing and protecting; the remaining gaps are especially `vessel + connecting` and fragile `vessel + remembering`.

## Discovery posture

Good candidates should support vessels as meaningful ritual carriers: cup, bowl, jar, bottle, cauldron, basin, water, offering vessel, shared object, altar vessel, or kept household object. The best sources should provide ritual mechanics rather than only ingredient correspondences.

Prioritize sources that may support:

- shared cup / shared bowl / shared object practice;
- household altar vessels;
- cup, bowl, basin, cauldron, water, or offering vessel rites;
- vessel as keeper of memory or witness;
- vessel as blessing / connection carrier;
- moon water or charged water where source-backed;
- keeping/holding/marking practices beyond protection bottles.
