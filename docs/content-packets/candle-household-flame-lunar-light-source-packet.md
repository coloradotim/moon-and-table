## Product Purpose

Jessica values candle magic and rituals around lunar and astrological phases/contacts. Moon & Table already has useful timing machinery and some Candle or light patterns, but the candle/lunar layer still needs richer ritual technology: what the light does, how it is activated, how it is lowered or closed, how it pairs with word/vessel/window/table/threshold, and how a timing contact changes the ritual's body instead of only changing the explanation.

This packet should not make Moon & Table louder, witchier, or more encyclopedic. It should make candle/light practices more materially real while preserving the product's private household-grimoire feel.

## Research Verdict

The strongest near-term path is not to import modern candle-spell manuals. The safest and most product-native path is:

1. Use existing approved household fire/light material first, especially Carmina Gadelica mechanics around kindling, smooring/banking, house/hearth/floor/household as one field, and new moon first-sighting.
2. Use NASA only for honest lunar visible-light timing: new/dark, waxing, full, waning, earthshine, and the phase cycle.
3. Treat lunar-material forms such as moonlit vessels, moonlit objects, moon water, and window bowls as dependent on #171 unless the form only uses already-approved visible-light timing and household-light mechanics.
4. Treat candle + written word as a bridge between this packet and #205/#175-style word mechanics. Do not copy charms, prayers, petitions, or spells.
5. Treat practical fire-safety sources as quiet eligibility/variant anchors only. Safety should never become the ritual's metaphysics or normal headline.
6. Defer candle colors, oils, herbs, burn readings, flame omens, love/protection/prosperity systems, and correspondence tables until Tim explicitly approves a separate source review.

Blunt read: Moon & Table can support strong candle magic soon, but it should do so through **household light as ritual action** rather than through imported candle-spell catalogs.

### Alexander Carmichael, _Carmina Gadelica, Volume I_

- Source title: _Carmina Gadelica, Volume I_
- Author / publisher: Alexander Carmichael; 1900 source family; Internet Sacred Text Archive edition
- Source type: Public-domain folk poetry, charms, blessings, household ritual material
- Reference URLs:
  - Kindling the Fire: <https://sacred-texts.com/neu/celt/cg1/cg1091.htm>
  - Smooring the Fire: <https://sacred-texts.com/neu/celt/cg1/cg1092.htm>
  - The New Moon: <https://sacred-texts.com/neu/celt/cg1/cg1057.htm>
  - Consecration of the Seed: <https://sacred-texts.com/neu/celt/cg1/cg1096.htm>
- Candidate status: `approved_candidate` / `adaptation_candidate`; direct mechanics for some household fire actions
- Why this belongs: It is already in the approved Batch 1 source set and gives Moon & Table the best source-backed household-light mechanics: kindling, banking/smooring, preserving light through rest, first-sighting of the new moon, seed/water preparation, and home-as-field logic.
- May support: first light; banked light; covered/lowered light; evening closure; beginning held until morning; household light connecting table/floor/threshold/room; new moon acknowledgment; seed/water beginning adjacent to lunar timing.
- Should not extract: prayers, hymns, Gaelic language/cadence, Christian formulae, saints/deity invocation, sectarian metaphysics, cultural reenactment, protection guarantees, copied phrasing.
- Source risks: The source is culturally specific and prayerful. Implementation must transform household-light mechanics into original private-household ritual language.
- Human-review notes: This is the strongest active foundation for Candle/light Batch A.
